# 🌦️ Task 3 - Weather App

## 📌 Project Overview
The Weather App is a web application built using HTML, CSS, and JavaScript. It allows users to search for any city and view real-time weather information using the OpenWeather API.

---

## 🚀 Features
- Search weather by city name
- Display current temperature
- Display weather condition
- Display humidity
- Real-time weather data from API
- Simple and responsive user interface

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API

---

## 📂 Project Structure

Task3_WeatherApp/
├── index.html
├── style.css
├── script.js
└── README.md

---

## 🎯 How to Run

1. Download or clone the project.
2. Open `script.js`.
3. Add your OpenWeather API key.
4. Open `index.html` in a browser.
5. Enter a city name and click Search.

---

## 📸 Output

The application displays:

- City Name
- Temperature (°C)
- Weather Description
- Humidity (%)

Example:

Mumbai

🌡️ Temperature: 31°C

☁️ Weather: Overcast Clouds

💧 Humidity: 69%
